# Código de Conducta y Ética

## Principios Generales
- **Integridad**: Actuar con honestidad y transparencia en todas las operaciones.
- **Respeto**: Mantener un ambiente de trabajo libre de discriminación, acoso o intimidación. Tolerancia cero ante comportamientos ofensivos.

## Uso de Recursos
- **Equipos informáticos**: Los portátiles y móviles de empresa son para uso profesional. Se permite un uso personal moderado y razonable que no interfiera con el trabajo ni comprometa la seguridad.
- **Internet**: Prohibido el acceso a sitios web con contenido ilegal, ofensivo o inapropiado desde la red corporativa.

## Confidencialidad
- Toda información sobre clientes, estrategias, precios y datos de empleados es estrictamente confidencial.
- No compartir contraseñas ni credenciales de acceso.
- Bloquear siempre el equipo al ausentarse del puesto de trabajo.

## Vestimenta (Dress Code)
- **Oficina**: "Business Casual". Se permite ropa cómoda pero profesional. No se permiten pantalones cortos deportivos, chanclas o camisetas con mensajes ofensivos.
- **Reuniones con clientes**: Se espera vestimenta formal acorde al contexto del cliente.

## Relaciones Personales
- Se permiten relaciones sentimentales entre empleados, siempre que no exista una relación de subordinación directa (jefe-empleado) para evitar conflictos de interés.
- Deben notificarse a RRHH si pueden afectar a la dinámica del equipo.
